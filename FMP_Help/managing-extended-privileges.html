创建和编辑扩展权限

扩展权限决定了文件的权限集所允许的数据共享选项。例如，如果该文件是共享的，那么"通过 FileMaker 网络访问"扩展权限将决定该权限集是否允许以客户端身份打开共享文件。"通过 FileMaker WebDirect 访问"扩展权限决定权限集是否允许从 Web 浏览器访问文件。请参阅关于帐户、权限集和扩展权限。

要创建和管理文件的扩展权限，必须使用被授予的权限集具有管理扩展权限权限的帐户访问权限来打开文件。如果没有使用此权限集打开文件，那么"文件"菜单 >"管理">"安全性"命令将为禁用状态。请参阅编辑其他权限。

可以在"高级安全设置"对话框中查看扩展权限，并为您希望允许访问的权限集启用每个扩展权限。默认情况下，除 fmreauthenticate10 以外的所有扩展权限都被禁用，即使在"完全访问权限"权限集中也是如此。

要创建或编辑扩展权限：

1. 选择"文件"菜单 >"管理">"安全性"。

如果"高级安全性设置"对话框中出现"扩展权限"选项卡，则跳至步骤 3。

2. 单击高级安全性设置，然后单击扩展权限选项卡。

3. 单击"新建"，或选择一个关键字，然后单击"编辑"。

4. 在"编辑扩展权限"对话框中，输入或更改扩展权限的相应关键字和描述。

5. 在"访问权限"区域中，选中或清除要对新权限集启用或禁用的每个权限集的复选框。

注释 

•启用扩展权限只是允许特定权限集访问共享数据。要实际访问共享数据，您还必须为所需的访问类型设置共享。请参阅在网络上共享文件、在 FileMaker Pro Advanced 中使用 ODBC 和 JDBC 或在 Web 上发布数据库。

•凭借 fmreauthenticateX 扩展权限，可以指定用户重新登录之前所需的时间。在"编辑扩展权限"对话框中的"关键字"区域，您可以：

•接受默认时间，即 fmreauthenticate10（10 分钟）

•通过编辑 fmreauthenticateX 末尾的数字，更改默认时间（最多可为 10080 分钟）

•创建新的 fmreauthenticate10 扩展权限，并指定不同的时间
