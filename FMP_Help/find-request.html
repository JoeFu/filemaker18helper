创建查找请求

要使用搜索条件查找记录，在查找请求的字段中键入搜索条件（要查找的值），该查找请求看起来像一条空白记录。在执行查找时，FileMaker Pro Advanced 会搜索符合您输入的条件的记录。匹配记录会替换以前任何的搜索结果。

要创建一个查找请求：

1. 从"布局"弹出菜单的查找模式中，选择包含要搜索的字段的布局。

2. 在查找请求下，选择要用于查找的字段，然后在该字段中键入值。

可以使用布局栏中的"插入运算符"列表来帮助您输入条件。例如，搜索条件 Jo@n 可用于查找名字 John 和 Joan。有关使用运算符的更多信息，请参阅查找文本和字符。

可以单击"省略"以排除匹配一组特定条件的记录。

3. 单击状态工具栏中的"执行查找"，或选择"请求"菜单 >"执行查找"。

查找请求会返回记录的搜索结果。状态工具栏中饼图 的突出显示区域显示占所有记录的比例。

提示

•您可以单击饼图在搜索结果与数据库中省略的记录之间进行切换。

注释 

•要在查找模式下访问最近查找条件的列表，请单击"已保存的查找条件"。或者在浏览模式下，单击"查找"旁边的向下箭头 (Windows)，或单击并按住"查找"(macOS)。然后从"最近的查找条件"列表中选择查找请求。

•要取消尚未完成的查找操作并保留以前的搜索结果不变，请按 Esc 键 (Windows) 或 Command-.（句点）(macOS)。

•要在浏览模式下显示所有记录，请单击显示全部。或者在查找模式下，选择"请求"菜单 >"显示所有记录"。（在执行查找之前，无需选择"显示所有记录"。FileMaker Pro Advanced 始终搜索您指定的表中的所有记录，除非您缩小了现有搜索结果。

•在未编制索引字段（例如参考相关字段的计算字段）上查找比在已建立索引的字段上查找需要的时间更长。请参阅定义字段索引选项。

•如果字段的值已建立索引，则可以使用该索引在查找请求中输入值。请参阅使用字段索引。

•您不能在容器字段、合计字段、全局字段或 Web 查看器中输入搜索条件。

•要在容器字段中查找数据，请创建可说明或标识容器字段内容的文本或数值字段。然后在该字段上执行查找。请参阅使用容器字段中的数据。

•当您在相关字段中执行查找时，FileMaker Pro Advanced 将显示有相关的记录与您输入的条件相匹配的所有记录。例如，在发票文件（其行项目在入口中显示为相关记录）中，通过在入口中的"项目"字段中键入计算机，您可以查找列出计算机的所有发票。

•要删除最近的查找条件，请单击"查找"旁边的向下箭头或单击"已保存的查找条件"(Windows)，或单击并按住"查找"或单击"已保存的查找条件"(macOS)。然后从列表中选择"清除所有最近的查找条件"。

•在查找模式下，字段中的此标记 用于确认您处于查找模式。（它不会出现在复选框、弹出菜单或单选按钮中。）
