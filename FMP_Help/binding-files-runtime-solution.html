将文件绑定至 Runtime 解决方案

注释  选中"使用高级工具"常规首选项时，可以使用此功能。请参阅使用高级工具。

使用"开发工具"，可以生成独立的 Runtime 数据库解决方案，用户无需运行 FileMaker Pro Advanced 即可访问该解决方案。"开发工具"可创建一份文件副本，并将一个或多个数据库文件绑定至您指定名称的 Runtime 应用程序。

要将数据库文件绑定至 Runtime 数据库解决方案：

1. 关闭所有您要自定义的数据库文件。

2. 选择"工具"菜单 >"开发工具"。

3. 如果您曾对同一数据库使用了"开发工具"并保存设置，则单击"加载设置"。

4. 单击"添加"，找到您要自定义的文件。

5. 如果添加多个文件，请双击列表中的文件以指定主文件。

6. 根据需要选择"开发工具"设置。请参阅针对开发人员解决方案设置文件。

7. 要显示"解决方案选项"，请单击"指定"。

8. 在"指定解决方案选项"对话框中，选择"创建 Runtime 解决方案应用程序"。

9. 对于"Runtime 名称"，键入 Runtime 应用程序的名称。

Runtime 名会用作 Runtime 应用程序文件名，以及包含 Runtime 数据库解决方案文件的文件夹名称。

10. 对于扩展名，请键入 1 到 5 个字符组成的文件扩展名，或使用默认扩展名 .fmpur。

该扩展名可用来将解决方案文件与 Runtime 应用程序相关联。请参阅命名 Runtime 解决方案。

11. 对于"绑定密钥"，键入 1 到 24 个字符长度的密钥。

重要事项  绑定密钥区分大小写。

绑定密钥会将 Runtime 应用程序链接到数据库文件，并确保绑定的文件仅在相应的 Runtime 应用程序中打开。

重要事项  绑定会安装与每个平台相关的系统文件。请参阅准备 Runtime 解决方案的文件。

12. 要将公司标志或其他自定义图像添加到关闭引导画面，请单击"指定"，并选择关闭图像，然后单击"选择"。

图像至少应为 382 x 175 像素 (72 dpi) 或更高，否则在显示时会扭曲。支持的图像格式为 JPEG 和 GIF。

13. 对于"延迟"，设置要引导画面显示的秒数。

单击"预览"，可以预览自定义引导画面的效果。

14. 根据需要选择其他选项。请参阅指定开发人员解决方案选项。

15. 单击"确定"。

16. 为了能够快速重复这个过程，请单击"保存设置"，并为设置文件选择文件夹和位置。请参阅保存解决方案设置。

17. 单击"创建"。

注释 

•设置绑定密钥：

•如果稍后需要将辅助文件添加到现有的 Runtime 数据库解决方案，则使用相同的密钥重新绑定文件。

•记住所使用的绑定密钥，并在安全的位置记录密钥。(您可以通过保存"开发工具"设置文件来记录密钥。请参阅保存解决方案设置）如果您忘记了绑定密钥，但要更改 Runtime 数据库解决方案，则需要使用新的绑定密钥重新绑定所有数据库文件，然后重新分发整个解决方案，包括新的 Runtime 应用程序。

•您可以在 FileMaker Pro Advanced 中打开绑定的 Runtime 文件进行修改。但是，如果在绑定文件时选择"从文件中永久地删除管理员访问权限"选项，则无法重新获得某些功能的访问权限。请参阅删除数据库的管理员访问权限。在这种情况下，您需要在 FileMaker Pro Advanced 中打开原始数据库文件，以更改设计，然后再使用分配给该 Runtime 数据库解决方案的绑定密钥重新绑定文件。
