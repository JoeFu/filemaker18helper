配置 ODBC 客户端驱动程序

无论您在关系图中导入 ODBC 数据，还是使用 ODBC 表，对于客户端应用程序而言，您都必须为所使用的 ODBC 数据源配置驱动程序。例如，如果要访问 Oracle 数据库中的记录，您将配置 Oracle 客户端驱动程序。

您与数据源交互、提供密码以及执行和显示查询结果的方式取决于每个应用程序的客户端驱动程序。此外，数据源配置可能随不同的 ODBC 驱动程序制造商而有所不同。

FileMaker Pro Advanced 支持的数据源

作为 ODBC 客户端应用程序，FileMaker 支持外部 SQL 数据源，如 Oracle、Microsoft SQL Server 和关系图中用作 ODBC 表的 MySQL Community Edition。如果您要导入 ODBC 数据或使用"导入记录"或"执行 SQL 语句"脚本步骤，就可以使用更多数据源。有关受支持的外部 SQL 数据源的信息，请搜索 FileMaker 知识库。有关使用 ODBC 数据源的各种方法的概览，请参阅在 FileMaker Pro Advanced 中使用 ODBC 和 JDBC。

注释  在 Windows 中，如果使用 32 位版本的 FileMaker Pro Advanced，则必须使用 32 位 ODBC 客户端驱动程序；如果使用 64 位版本的 FileMaker Pro Advanced，则必须使用 64 位 ODBC 客户端驱动程序。在 macOS 中，必须使用 64 位 ODBC 客户端驱动程序。

将以下过程用作配置数据源的常规指导原则（详细信息，请参考每个数据源应用程序附带的文档）。

窗口

1. 启动"ODBC 数据源管理器"。

在控制面板中，选择"系统和安全">"管理工具"然后，执行以下某项操作：

•32 位 Windows：打开"数据源 (ODBC)"。

•64 位 Windows：

 

对于 FileMaker Pro Advanced

请执行此操作

32 位

•Windows（64 位）：打开"数据源 (ODBC)（32 位）"。

•Windows 7（64 位）：双击 C:\Windows\SysWOW64 文件夹中的 odbcad32.exe 文件。

64 位

•Windows（64 位）：打开"数据源 (ODBC)（64 位）"。

•Windows 7（64 位）：打开"数据源 (ODBC)"。

2. 单击"系统 DSN"选项卡或"用户 DSN"选项卡。

如果您的数据源出现在该列表中，则表明已配置了驱动程序。您可以跳过其余的步骤并开始访问 ODBC 数据，或者选择数据源，然后单击"配置"来修改与数据源的交互方式。

"系统 DSN"（数据源名称）对登入到电脑的所有用户均可用。只有在登录时用户 DSN 才可用。

重要事项  使用关系图中的 ODBC 表时，仅支持"系统 DSN"。

3. 单击"添加"，为您的数据源配置驱动程序。

如果没有出现您要使用的数据源的驱动程序，则可以使用来自第三方供应商的驱动程序。

要安装新的驱动程序，请使用驱动程序的安装程序。

4. 选择要使用 FileMaker Pro Advanced 访问的数据源的驱动程序，然后单击"完成"。

将显示设置对话框。

5. 在"名称"中，输入描述性的文本来标识数据源。

许多驱动程序还提供了用于自定义数据源访问方式（如标识一个特殊的数据源文件）的选项。输入描述性文本（如果需要）。

6. 单击"完成"。

7. 单击"确定"以保存数据源信息。

macOS

1. 从 Actual Technologies 中安装 ODBC Manager（下载网址为 www.odbcmanager.net）。

注释  ODBC Manager 是不受 FileMaker 支持的免费软件产品。

2. 启动 ODBC 管理程序实用工具。

3. 单击"系统 DSN"选项卡（或者，对于 ODBC 导入，单击"系统 DSN"选项卡，或"用户 DSN"选项卡）。

重要事项  使用关系图中的 ODBC 表时，仅支持"系统 DSN"。

如果您的数据源出现在该列表中，则表明已配置了驱动程序。您可以跳过其余的步骤并开始访问 ODBC 数据，或者选择数据源，然后单击"配置"来修改与数据源的交互方式。

"系统 DSN"（数据源名称）对登入到电脑的所有用户均可用。只有在登录时用户 DSN 才可用。

4. 单击"添加"，为您的数据源配置驱动程序。

如果没有出现您要使用的数据源的驱动程序，则可以使用来自第三方供应商的驱动程序。

要安装新的驱动程序，请使用驱动程序的安装程序。

5. 选择要使用 FileMaker Pro Advanced 访问的数据源的驱动程序，然后单击"确定"。

6. 在配置对话框中，单击"继续"。

7. 在"名称"中，输入描述性的文本来标识数据源。

许多驱动程序还提供了用于自定义数据源访问方式（如标识一个特殊的数据源文件）的选项。输入描述性文本（如果需要）。

8. 单击"完成"。

9. 单击"完成"来保存您的数据源信息。

配置驱动程序后，您便可以访问数据源中的记录。请参阅连接外部数据源和查询来自 FileMaker Pro Advanced 的 ODBC 数据源。
