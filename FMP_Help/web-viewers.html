在布局上使用 Web 查看器

可以使用 Web 查看器直接在 FileMaker Pro Advanced 布局中显示 Web 页。Web 查看器是布局对象，使您可以根据数据库中的数据显示网站信息。例如，Web 查看器可以根据数据库表的地址字段和城市字段中的信息计算地图网站的 URL。当您浏览表中的每条记录时，Web 查看器将显示当前记录中指定位置的地图。

Web 查看器使用操作系统的 Web 浏览器技术，因此您可以执行很多可以用 Web 浏览器执行的任务。可以单击链接并导航到 Web 查看器最初加载的页面以外的页面。Web 查看器还可以显示网站中的内容，如 PDF 文档和 QuickTime 电影（您可能需要为其安装其他软件）。您也可以控制 URL 编码。

为便于您创建 Web 查看器，FileMaker Pro Advanced 提供了多个数据驱动网站的模板。在使用其中的某个模板时，您只需指定网站使用的参数。

还可以为任何网站通过定义计算来创建自定义 Web 地址。FileMaker Pro Advanced 解析表达式以构造一个 URL，Web 查看器将使用此 URL 显示 Web 页。

Web 查看器并不是为了取代 Web 浏览器而设计的。对于某些网站，您可能需要打开一个单独的 Web 浏览器窗口。

要添加或更改 Web 查看器：

1. 在布局模式下，从"布局"弹出菜单中选择所需的布局。

2. 单击状态工具栏中的"Web 查看器工具"，然后在布局上拖动十字线以绘制 Web 查看器。

要更改 Web 查看器，请在布局上双击 Web 查看器。然后跳到步骤 4。

3. 从"选择一个网站"列表中选择一个网站。

要定义自定义 Web 地址，请从列表中选择"自定义 Web 地址"。单击"Web 地址"框旁边的"指定"，然后为该 web 地址定义一个计算并单击"确定"。（请参阅使用公式和函数。）然后跳到步骤 5。

4. 对于要使用的每个网站参数，请执行下列任意操作：

 

目的

请执行此操作

添加或更改包含该参数的字段

单击 并选择"指定字段"。然后选择一个数据库字段并单击"确定"。或者，直接在网站参数框中键入字段。

添加或更改生成该参数的计算

单击 并选择"指定计算方式"。然后创建或更改计算公式并单击"确定"。

指定一个常数值

删除用于网站参数的字段或计算方式

在网站参数框中，选择项目，然后按 Backspace 键或 Delete 键。

5. 选择 Web 查看器的选项，例如显示进程条或错误消息和安全状态。

对于"在查找模式下显示内容"，用户在字段中输入值并退出该字段后，如果 Web 地址基于该字段，则 Web 查看器将重新计算 URL 并重新加载 Web 页。

对于"自动编码 URL"，FileMaker Pro Advanced 根据需要将编码规则应用到 URL，以便遵循浏览器所需的格式。请参阅关于 Web 查看器中的 URL 编码。

注释 

•Web 查看器在表单视图和列表视图中显示 Web 页。在"表视图"中，Web 查看器只在位于布局的页首、页脚部分或者顶部或底部导航部分时才会显示。

•无法将 Web 查看器放在入口中。如果将 Web 查看器放在入口中，则 Web 查看器将在布局上显示为一个与入口重叠的对象。您可以将 Web 查看器放在入口内的弹出框中，但 Web 查看器的上下文为该入口的第一条记录，而非该入口的当前记录。

•不要将对象置于 Web 查看器之前或之后。在浏览、查找和预览模式下，Web 查看器无法按照您在布局模式下设置的堆迭顺序显示。

•通过将数据嵌入到一个 URL，可以将 html 数据发送到 Web 查看器。请参阅设置 Web 查看器脚本步骤。

•要查看基于自定义 Web 地址计算的 URL，在布局模式下选择"查看"菜单 >"显示">"样例数据"。

•必须确保自定义 Web 地址：

•以正确的方案开头，例如 http://、https://、ftp:// 或 file://

•正确编码为 URL，方式是通过选择"自动编码 URL"
