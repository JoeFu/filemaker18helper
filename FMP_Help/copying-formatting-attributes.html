要使用"检查器"将格式属性和自定义样式应用到其他布局对象、部分或背景：

在布局模式下，单击"检查器"中的"外观 "选项卡。然后：

 

目的

请执行此操作

拷贝对象所有状态的格式或自定义样式或者应用到布局部分的格式或自定义样式

选择对象或部分标签，然后单击对象类型列表（位于选项卡顶部）右侧的 。

拷贝对象的选定显示状态的格式属性

选择对象，从对象状态列表中选择要拷贝的显示状态，然后单击列表（位于选项卡顶部）右侧的 。

有关显示状态的更多信息，请参阅指定对象的显示状态。

拷贝布局背景的格式属性

单击布局背景内任意位置将其选中，然后单击对象类型列表（位于选项卡顶部）右侧的 。

粘贴格式属性

选择一个或多个对象，然后选择要向其应用拷贝的格式属性的显示状态。（您可以选择不同于您拷贝的显示状态的显示状态。）单击对象状态列表右侧的 。

粘贴您拷贝的对象的所有显示状态的格式属性或自定义样式

选择一个或多个对象，然后单击对象类型列表右侧的 。

粘贴您拷贝的布局部分的格式属性或自定义样式，或者粘贴布局背景的格式属性

选择部分标签或单击布局背景内任意位置将其选中，然后单击对象类型列表右侧的 。

注释  不能将布局背景的自定义样式从一个布局粘贴到另一个布局；只能粘贴格式属性。

注释 

•对于有多个组件的对象，每个组件都有一个单独的样式。例如，入口具有"入口"样式，该样式可设置整个对象的格式，而"入口：行"样式则仅格式化入口的行。对于具有多个组件的对象，您仅可以拷贝和粘贴用于整个对象的格式（在这种情况下，是"入口"而不是"入口：行"）。整个对象的样式始终都是对象类型列表（位于检查器中的"外观 "选项卡顶部）中列出的第一个样式。

•要在对象之间拷贝和粘贴自定义样式，对象必须具有相同类型或相似类型，如矩形和椭圆形，而且必须在同一布局上。如果对象不具有相同类型，那么会粘贴对象可以使用的任何格式属性，但目标对象上的样式名称不会改变。

•如果将弹出框按钮拷贝到其他 FileMaker Pro Advanced 解决方案，弹出框在该解决方案中相同，但 FileMaker Pro Advanced 不会将应用于弹出框按钮的任何样式拷贝到该解决方案。

要使用"格式刷"工具将格式属性应用到布局对象、部分或背景之间：

1. 在布局模式下，选择对象、部分标签，或单击具有您想要应用到其他对象、部分或背景的格式属性的布局背景中的任意位置。

注释  要选择布局背景，请确保未选中任何对象或部分。

2. 单击状态工具栏中的"格式刷工具 "，或选择"格式"菜单 >"格式刷"。

提示 要拷贝对象的显示状态，请按住 Alt 键 (Windows) 或 Option 键 (macOS)，然后单击"格式刷工具 "。

3. 执行以下某项操作：

 

要将格式属性应用于

请执行此操作

单个对象或布局部分

选择要更改的对象或部分标签。

多个对象

拖动箭头指针形成一个包含这些对象的选择框。选择框不必完全包围对象。要避免包含部分选定的对象，请在拖动时按 Ctrl 键 (Windows) 或 Command 键 (macOS)。

或者，要将拷贝的格式单独应用于多个对象，请双击"格式刷"工具将其锁定，然后按住 Shift 键并单击每个对象。要解除"格式刷"工具锁定并将其关闭，请按 Esc 键。

布局背景

从"布局"弹出菜单选择不同的布局，然后单击其布局背景。

对于要显示的新布局背景，一个或多个布局部分必须为透明。请参阅设置填充、线条样式、边框、阴影和内边距。
