文本函数

文本函数可用于分析、重新排列、提取和生成字符串。例如，您可以使用 MiddleWords 函数从提供的文本中提取特定的单词。

文本函数使用如下参数：

•文本类型字段

•文本常数（引号内）

•具有文本结果的表达式

 

此函数

执行操作

Char

返回数字中 Unicode 代码点的字符。

Code

返回文本中字符的 Unicode 代码点。

Exact

如果任意两个字段的内容匹配，则返回"1"(真)；否则返回"0"(假)。

Filter

只返回"要筛选的文本"中的那些在"筛选文本"中指定的字符（按这些字符在"要筛选的文本"中的原始输入顺序返回）。

FilterValues

返回只包含"筛选值"中所提供的值的文本结果（按这些值在"要筛选的文本"中的原始输入顺序返回）。

GetAsCSS

返回转换成 CSS（层叠样式表）格式的文本。

GetAsDate

以字段类型日期的形式返回文本格式的日期。

GetAsNumber

以字段类型数值的形式仅返回文本格式的数字。

GetAsSVG

返回转换成 SVG（可缩放向量图形）格式的文本。

GetAsText

以字段类型文本的形式返回数据。

GetAsTime

以字段类型时间的形式返回文本格式的时间或时间戳。

GetAsTimestamp

以字段类型时间戳的形式返回文本。

GetAsURLEncoded

返回 URL（统一资源定位器）编码形式的文本，以用作 URL。

GetValue

返回"值列表"中由"值个数"指定的所需值。

Left

返回文本中指定的"字符个数"的文本（从左计数）。

LeftValues

返回文本中包含的值，根据指定的"值个数"，从左计数。

LeftWords

返回文本中包含指定的"字数"的文本（从左计数）。

Length

返回字段中的字符个数，其中包括所有空格、数字和特殊字符。

Lower

以小写形式返回指定文本中的所有字母。

Middle

从文本中提取指定的"字符个数"，从指定的字符位置开始。

MiddleValues

返回文本中包含指定"值个数"的文本，从"开始值"处开始。

MiddleWords

从文本中返回包含指定"字数"的文本，从"开始字"处开始。

PatternCount

返回文本中"查找的字符串"的匹配项数。

Position

返回文本中"查找的字符串"的指定匹配项所在的开始位置。

Proper

以每个单词的首字母大写、其他字母小写的形式返回文本中的单词。

Quote

返回由引号括起来的文本的文本格式。

Replace

将文本中的一串字符替换成"替换文本"。

Right

返回文本中指定的"字符个数"（从右计数）。

RightValues

返回文本中包含的值，根据指定的"值个数"，从右计数。

RightWords

返回文本中包含指定"字数"的文本（从右计数）。

SerialIncrement

返回指定的文本和数字，数字按指定数量递增。

SortValues

根据指定的数据类型和区域设置为一列值排序。

Substitute

返回文本字符串，并将指定文本中"查找的字符串"的每个匹配项都替换为"替换字符串"。

Trim

返回除去所有前置和后置空格的文本。

TrimAll

返回一个除去或插入指定空格的文本副本。

UniqueValues

根据指定的数据类型和区域设置返回列表中的唯一值。

Upper

以大写形式返回指定文本中的所有字母。

ValueCount

返回指定文本中值的总数。

WordCount

返回指定文本中字的总数。
