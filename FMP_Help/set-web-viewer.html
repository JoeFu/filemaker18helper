设置 Web 查看器

控制指定的 Web 查看器。

另请参阅 

在布局上使用 Web 查看器

使用脚本自动执行任务

选项 

•"对象名称"是要使用的 Web 查看器的名称。请参阅命名对象。

•"操作"可让您选择下列某个选项：

•"重置"将指定的 Web 查看器重置到它最初指定的 Web 地址。该操作还会清除此 Web 查看器的"后退"和"前进"历史记录。

•"重新加载"将重新加载 Web 查看器正在显示的网页。

•"前进"将前进一页，方式与 Web 浏览器相同。

•"后退"将后退一页，方式与 Web 浏览器相同。

•"转到 URL"可以让您指定一个要在 Web 查看器中加载的新 Web 地址。新 Web 地址是您在"设置 Web 查看器 - 转到 URL"选项对话框中指定的算式。在此脚本步骤中，您无法更改 Web 查看器是否允许交互，显示进程条或状态信息，在查找模式下显示内容，或自动编码 URL。仅当在布局模式下添加或更改 Web 查看器时，才能设置这些选项。

说明 

此脚本步骤仅适用于指定的 Web 查看器对象。

要指定对象名称，请在"对象名称"框中键入名称，或单击"指定"并创建一个算式。

如果选择"转到 URL"操作：

•使用在布局上创建 Web 查看器的相同方法来指定网址。有关更多信息，请参阅在布局上使用 Web 查看器。

•可以使用以下格式将 html 数据包括在 URL 中，以将该数据发送到 Web 查看器：

data:[<mediatype>][;base64],<data>

其中应用了以下语法：

dataurl = "data:" [ mediatype ] [ ";base64" ] "," data

mediatype = [ type "/" subtype ] *( ";" parameter )

data = *urlchar

parameter = attribute "=" value

有关"数据 URL 方案"的更多信息，可以在 Web 上找到。

注释 

•FileMaker Pro Advanced 仅支持 UTF-16 字符编码。使用其他方法编码的字符数据不会在 URL 的目标中显示。

•FileMaker WebDirect 不支持"前进"和"后退"选项。

示例 1 

将名为"产品网站"的 Web 查看器重置为其最初指定的 Web 地址。

设置 Web 查看器 [对象名称: "产品网站"; 操作: 重置]

示例 2 

在"产品网站"Web 查看器中显示 FileMaker, Inc. 主页：

设置 Web 查看器 [对象名称: "产品网站"; URL: "http://www.filemaker.com"]

示例 3 

显示在"产品网站"Web 查看器的"网站"字段中输入的 URL：

设置 Web 查看器 [对象名称: "产品网站"; URL: 产品::网站]

示例 4 

使用数据 URL 方案在名为 WV2 的 Web 查看器中显示一个小图标。

设置 Web 查看器 [对象名称: "WV2"; URL: "data:image/gif;base64,R0lGODlhFwAMAKEAAL+/v///AAAAUwAAACH5BAEAAAAALAAAAAAXAAwAAAI7hBGHapHcXJKPumizpigI+QliSH0XIjokWJ6oB4+qt0Zmaqpjesxz7st1YD8ZEbhJajAuDgfSYTx60wIAOw=="]
