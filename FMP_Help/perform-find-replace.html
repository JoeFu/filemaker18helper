执行查找/替换

根据"执行查找/替换"选项对话框中的选项查找/替换数据。

另请参阅 

查找和替换数据

使用脚本自动执行任务

选项 

•"包括对话框"指定在查找/替换操作结束时是否显示"查找/替换总计"对话框。将"包括对话框"设置为"关闭"时，还会阻止在执行"全部替换"操作时显示确认对话框。

如果希望用户能够输入查找或替换的条件，请使用打开查找/替换脚本步骤。

•"执行"可让您选择希望查找/替换执行的操作：

•"查找下一个"用于查找搜索条件实例。

•"查找和替换"用于查找搜索条件实例，然后替换条件，再次运行脚本时将搜索下一个实例。

•"替换"将使用替换条件替换与查找条件相匹配的高亮显示文本。使用"替换"操作时，您可能需要组合使用"执行查找/替换"脚本步骤与全选脚本步骤。

•"全部替换"用于查找和替换查找条件的所有实例。

•"查找内容"指定查找条件。

•"替换为"指定用于替换查找条件实例的文本。

•"方向"指定您希望查找/替换搜索记录的方向。

•"匹配大小写"在执行查找时将考虑查找条件的大小写。

•"全字匹配"将忽略查找条件嵌入在其他词语中的情况。

•"所有记录/请求"将在搜索结果的所有记录/请求中进行查找。

•"当前记录/请求"仅限在当前记录/请求中进行查找。

•"所有字段"将对布局中的每个字段进行查找。

•"当前字段"仅限在活动字段中进行查找。可以使用转到字段脚本步骤在执行查找/替换前选择字段。

说明 

如果"执行查找/替换"未找到与查找条件匹配的数据，则不会返回错误。

示例 1 

替换所有记录中与查找条件相匹配的"再订购标准"字段中的文本，而不提示用户。

转到字段 [产品::再订购标准]
执行查找/替换 [包括对话框: 关闭; 1; 3; 全部替换]

示例 2 

转到"描述"字段中产品名称的下一个匹配项。

显示自定义对话框 ["您是否要在"描述"字段中查找当前产品名称?"]
If [Get (上次选择信息) = 1]
转到字段 [产品::描述]
执行查找/替换 [包括对话框: 关闭; 产品::名称; 查找下一个]
End If

示例 3 

用记录数据替换当前字段中的特定条件。

显示自定义对话框 ["您是否要用记录数据替换占位符条件?"]
If [Get (上次选择信息) = 1]
执行查找/替换 [包括对话框: 关闭; "[名称]"; 客户::名称; 全部替换]
执行查找/替换 [包括对话框: 关闭; "[公司]"; 客户::公司; 全部替换]
执行查找/替换 [包括对话框: 关闭; "[地址]"; 客户::地址; 全部替换]
End If
