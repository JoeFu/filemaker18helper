编辑其他权限

1. 在"编辑权限集"对话框中开始编辑新的或现有的权限集。

请参阅创建和编辑权限集。

2. 在"其他权限"区域中，设置以下一个或多个选项。

 

目的

请执行此操作

•允许打印。

•允许将记录另存为 PDF 文件。

•允许使用 Finder Apple 事件的"打印"来进行打印。

选择"允许打印"。

•允许导出记录。

•允许将记录另存为 Excel 文件。

•允许将搜索结果中的所有记录拷贝到剪贴板。

•允许从 FileMaker 文件导入记录。

•允许保存记录的副本。

•macOS：允许使用以下 Apple 事件查询数据：GetCellValue、字段内容、记录值、目录和布局内容。

选中"允许导出"。

注释  在 Windows 中，清除该选项对 ActiveX Automation 没有影响。在 macOS 上，清除该选项只会禁止访问数据的 Apple 事件，而其他 Apple 事件仍会继续工作。

•允许具有此权限集的用户在设置 FileMaker Network 共享、Web 发布和 ODBC/JDBC 共享（通过文件菜单 > 共享命令）时启用和禁用扩展权限。

•在选择文件菜单 > 管理 > 安全性时，允许拥有此权限集的用户在"高级安全性设置"对话框中编辑扩展权限。

选择管理扩展权限。要启用对"高级安全性设置"对话框的访问，还要将可用菜单命令设置为全部。

此权限不允许用户打开"管理安全性"对话框。

允许用户：

•在"管理安全性"对话框中查看所有帐户访问权限条目

•创建、修改和删除没有完全访问权限集的帐户访问权限条目

•分配除完全访问权限之外的任何现有权限集

选择管理没有完全访问权限的帐户，并将可用菜单命令设置为全部。

此权限不允许用户：

•将完全访问权限集分配给任何帐户访问权限条目或从任何条目中删除

•更改帐户访问权限条目优先级

•打开"高级安全设置"对话框，这意味着无法查看、创建、修改或删除权限集、扩展权限或文件访问权限的设置

允许为字段输入与验证选项集不匹配的数据。

选中"允许用户替代数据验证警告"。

注释  当将字段定义为禁止字段验证替代（通过清除"允许用户在数据输入过程中替代"验证选项）时，将忽略此设置。请参阅定义字段验证。

当共享文件的用户闲置时，断开该用户与 FileMaker Server 或 FileMaker Cloud for AWS 的连接。（这通常会提高 FileMaker Server 的性能。）

选择"闲置时断开用户与服务器的连接"。

注释  如果使用 FileMaker Server 或 FileMaker Cloud for AWS 托管共享文件，请参阅 关于断开空闲用户连接。

允许具有此权限集的用户更改其密码（仅限 FileMaker 文件帐户）。

选中"允许用户修改自己的密码"。

要求具有此权限集的用户在指定天数后更改其密码（仅限 FileMaker 文件帐户）。

（过去指定的天数后，每个用户下次打开该文件时，将会出现"更改密码"对话框。该用户必须更改密码才能打开该文件。）

选中"每隔 <n> 天后必须更改"，然后输入天数。

注释  如果选中此选项时没有同时选择"允许用户修改自己的密码"，那么具有完全访问权限的用户必须代替被授予此权限集的帐户更改该密码。（在大多数情况下，应该同时选中或清除这两个选项。）

指定新密码的最短字符长度（仅限 FileMaker 文件帐户）。

选中"最短密码长度"，然后输入字符数。

指定当文件活动时只有有限的菜单命令集可用。

请为"可用菜单命令"选择下列某个选项：

•"全部"：启用所有的菜单命令，由其他权限集选项明确禁用的除外。

•"只限于编辑"：只启用浏览模式中的基本编辑命令。

•"最小"：禁用所有的菜单命令，但以下命令除外：打开、关闭和创建数据库文件，切换数据库窗口，执行脚本、打开 FileMaker Pro Advanced 帮助或更改首选项选项。
