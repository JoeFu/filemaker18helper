从 FileMaker Pro Advanced 中导出数据

可以将 FileMaker Pro Advanced 数据导出到新文件，然后在其他应用程序中打开它。

如果 FileMaker Pro Advanced 源文件包含合计字段，则可以按排序字段进行分组以便导出小计值，例如由带分组数据的报表生成的小计值。该过程为每个组导出一条记录。例如，如果某个报表按区域生成销售数据总计，则可以为每个区域导出一个合计值。

还可以将字段内容导出到文件。请参阅导出字段内容。

要导出 FileMaker Pro Advanced 数据以用于其他应用程序：

1. 打开 FileMaker Pro Advanced 文件，并显示一个布局，其中显示了要导出的记录。

2. 查找要导出的记录并对其进行排序。

要导出分组小计值，请在排序顺序中加入条件字段，或者运行生成所需小计的报表。有关查找记录以及对记录排序的信息，请参阅查找记录和对记录排序。

3. 选择"文件"菜单 >"导出记录"。

4. 在"将记录导出到文件"对话框中，为该文件键入名称并选择一个位置。

如果要导出到 XML 并在导出过程中应用 XSLT 样式表，请为文件名指定一个与转换后的文件格式相对应的扩展名（如 .txt、.htm 或 .xml），以便不同的操作系统能够正确地识别该文件。

5. 从"保存类型"(Windows) 或"类型"(macOS) 中选择一个文件类型，然后单击"保存"。

请使用应用程序（即要在其中使用导出数据的应用程序）支持的文件格式。有关文件格式的信息，请参阅支持的导入/导出文件格式。

6. 对于"保存后"，选择一个选项。

7. 执行以下某项操作：

•如果要导出到 XML，则将显示"指定 XML 和 XSL 选项"对话框。继续执行步骤 8。

•如果要导出到任何其他文件类型，请跳到步骤 9。

8. 在"指定 XML 和 XSL 选项"对话框中，执行以下操作：

•通过选择 FMPDSORESULT 格式或 FMPXMLRESULT 格式，指定要使用的 XML 语法。

•要在导入过程中对 XML 应用 XSLT 样式表，请选择"使用 XSL 样式表"并指定 XSLT 样式表源。XSLT 源可以是文件，也可以是 HTTP 请求的结果。

9. 在"指定导出时的字段顺序"对话框中，指定希望 FileMaker Pro Advanced 如何导出数据。

•当"指定导出时的字段顺序"对话框打开时，表列表默认为"当前布局（布局名称）"，并只显示当前布局上的可见字段。要显示当前表中的所有字段名称，请从列表中选择"当前表（表名）"。

•如果当前布局包含面板控件，则"指定导出时的字段顺序"对话框中只显示当前面板上的字段。

 

目的

请执行此操作

在导出中包含字段

双击该字段名以将其移动到"字段导出顺序"区域中。

在导出中包含相关字段

从该对话框左上角的列表中选择相关表的名称，然后将所需字段移动到"字段导出顺序"区域。

导出分组合计值而不是单条记录

在左侧的字段列表中，单击要对其值进行汇总的合计字段。在"分组依据"区域中，选择每个作为值汇总依据的条件字段。然后单击"移动"。

单击"移动"时，FileMaker Pro Advanced 将合计字段和一个或多个分组合计字段添加到"字段导出顺序"区域。（FileMaker Pro Advanced 为您在"分组依据"区域中选择的每个条件字段均添加一个分组合计字段。）分组合计字段的语法为合计字段 by 分组依据字段，例如"销售总计 by 区域"。

注释  除非在开始导出之前已对数据进行排序，否则将无法导出分组合计值。（如果数据未经排序，则"分组依据"区域将显示"未排序"，而不显示任何字段。）

从"字段导出顺序"中删除字段并阻止导出其数据

对于"字段导出顺序"，请选择一个字段，然后单击"清除"。

更改字段导出顺序

对于"字段导出顺序"，指向字段名左侧的双箭头并上下拖动。

10. 如果必要，对于"输出文件字符集"，选择希望导出文件使用的字符编码。

11. 如有必要，请选择"将当前布局的数据格式应用于导出的数据"，以设置当前布局上出现的字段数据的格式。

FileMaker Pro Advanced 使您可以设置布局上字段的格式，这将使字段在布局上的显示状态不同于它在数据库文件中的实际存储格式。（请参阅在布局模式下设置字段对象的格式和属性。）如果要使用当前布局中指定的字段格式导出数值、日期和时间数据，请选择"将当前布局的数据格式应用于导出的数据"。

如果在导出到 Excel 文件时无法应用当前布局格式，则 Excel 文件会使用系统格式。

12. 单击"导出"。

注释 

•要将 FileMaker Pro Advanced 用作数据源以通过 ODBC 或 JDBC 提供 FileMaker Pro Advanced 数据，请参阅通过 ODBC 或 JDBC 共享 FileMaker Pro Advanced 数据。
