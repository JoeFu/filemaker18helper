管理数据库键盘快捷方式 (Windows)

在"管理数据库"对话框中工作时，可以使用以下键盘快捷方式。

字段类型键盘快捷方式

可以在"管理数据库"对话框的"字段"选项卡中使用这些键盘快捷方式。

 

要指定如下字段类型

按键

文本

Ctrl+T

数字

Ctrl+N

日期

Ctrl+D

时间

Ctrl+I

时间戳

Ctrl+M

容器

Ctrl+R

计算

Ctrl+L

合计

Ctrl+S

表和关系的选择与排列快捷方式

 

目的

按键

在关系图各处移动所选项，单独选择表、注释或关系

向上键、向下键、向左键、向右键

在对象间递增地移动所选项

Shift+向上键、Shift-向下键、Shift-向左键、Shift+向右键

将所选项移动到包括以特定字符开头的文本的对象上

键入字符

注释  如果多个对象共享初始字符，则可以通过按 Ctrl+G 或再次键入字符将选定内容移动到下一个匹配项。

在关系图和命令按钮间切换

Tab

移动所选对象

在选定对象后拖动或按 Ctrl+向上键、Ctrl+向下键、Ctrl+向左键、Ctrl+向右键

调整所选对象的大小

在选择了某个完全展开的表后按 Ctrl+Shift+向上键、Ctrl+Shift+向下键、Ctrl+Shift+向左键、Ctrl+Shift+向右键

选择多个表

按住 Shift 键并单击多个表或将选择指针拖过多个表

注释  要通过拖动并完全包围的方式选择表，请在拖动的同时按住 Ctrl。

选择所有表和注释

Ctrl+A

选择所有表

Ctrl+Shift+A

选择与当前选定的表直接关联的所有表

Ctrl+Y

选择数据源相同的表

Ctrl+U

切换选定的表或注释的显示状态

Ctrl+T

关系图快捷方式

 

目的

按键

添加表

Ctrl+Shift+T

编辑表

在选定一个表时按 Ctrl+O

删除表

在选定了一个或多个表时按 Delete 进行删除

添加关系

Ctrl+Shift+R

编辑关系

在选定一个关系时按 Ctrl+O

删除关系

在选定了一个或多个关系时按 Delete 进行删除

添加文本注释

Ctrl+Shift+N

复制所选项

Ctrl+D，或者在拖动的同时按 Ctrl

选择并保持对象选择光标的选中状态

Ctrl+R

选择并保持放大光标的选中状态

Ctrl+ =（等号）

注释  按住 Shift 的同时单击会临时选择视图缩小光标。

选择并保持视图缩小光标的选中状态

Ctrl+ -（连字号）

注释  按住 Shift 的同时单击会临时选择放大光标。

选择并保持注释工具的选中状态

Ctrl+N

更改关系图的显示比例

键入百分比值，然后按 Return

将选择项从关系图移动到命令按钮并从左向右移动

Tab

从右向左在命令按钮之间移动选择项，并在"添加表"命令处移动到关系图上

Shift+Tab

贴紧

Ctrl+I

打开或关闭页面指南

Ctrl+E

显示打印设置选项

Ctrl+Shift+P

选择另一个按钮

Tab、向左键、向右键

触发选定的命令按钮

空格键

在选定按钮的菜单中触发一个命令

按 Enter 或空格键显示菜单，按向上键或向下键在菜单中选择内容，然后用 Enter 键触发命令

撤销上一个命令

Ctrl+Z

恢复上一个命令

Ctrl+Shift+Z
