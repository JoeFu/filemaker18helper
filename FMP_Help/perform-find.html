执行查找

使用当前或存储的查找请求查找记录。

另请参阅 

创建查找请求

使用脚本自动执行任务

选项 

"指定查找请求"允许您创建和管理查找请求。所创建的请求将与该脚本步骤一起保存（恢复指示存储的查找请求）。有关更多信息，请参阅"指定查找请求"和"编辑查找请求"对话框。

•"新建"会打开"编辑查找请求"对话框，您可以从中定义查找请求的条件。

•"编辑"将打开列表中选定的查找请求。

•"复制"将从列表中复制一个或多个选定的查找请求。

•"删除"将从列表中删除一个或多个选定的查找请求。

"编辑查找请求"对话框使您可以创建或编辑查找请求的条件。

•对于"操作"，选择"查找记录"或"省略记录"，从而指定此查找请求将用于查找记录还是省略记录。查找记录将把记录添加到搜索结果中。省略记录将排除记录。单个请求可以查找或省略记录，如果需要在同一"执行查找"脚本步骤中查找和省略记录，则请使用多个请求。

•"查找记录的条件"（或"省略记录的条件"）将显示当前表中字段的列表。要构造查找请求，请首先从此列表中选择一个字段。

•要从相关表中选择字段，请单击列表顶部的当前表名称，然后选择所需的相关表。从这个新列表中选择相关字段。

•更改"重复项"中的值以指定重复字段的某个特定单元。

•在"条件"区域中，为选定字段键入搜索条件。

•单击"插入运算符"进一步细化搜索条件。请参阅查找记录。

•单击"添加"将条件添加到查找请求中。

•要更改现有条件，请从对话框顶部选择包含该字段及条件的行，然后更改字段和/或条件。单击"更改"存储您所做的更改。

•要删除现有条件，请从对话框顶部选择包含该字段及条件的行，然后单击"删除"。

说明 

使用"执行查找"时，可以在此脚本步骤中指定查找请求，也可以不指定该请求。

•如果您指定了一个或多个查找请求，此脚本步骤会进入查找模式，执行指定的查找请求，然后进入浏览模式。

•如果您未指定查找请求：

•在查找模式下，此脚本步骤执行当前查找请求，然后进入浏览模式。

•在任何其他模式下，此脚本步骤进入查找模式，执行上次执行的查找请求，然后进入浏览模式。

您可以在进入查找模式脚本步骤之后使用"执行查找"，暂停该脚本以允许用户在执行查找前输入查找条件或使用设置字段脚本步骤指定查找条件。

示例 1 

转至"发货单详情"布局，进入查找模式，暂停以便用户进行输入，执行查找，然后返回到原始布局。

转到布局 ["发货单详情"]
进入查找模式 [暂停: 打开]
执行查找 [ ]
转到布局 [原始布局]

示例 2 

转到查找模式，将"日期"字段设置为当前日期，并执行查找。显示"发货单详情"布局、"发货单"布局或自定义对话框，具体取决于使用当前日期找到的记录数量。

设置错误捕获 [打开]
进入查找模式 [暂停: 关闭]
设置字段 [发货单::日期; Get (当前日期)]
执行查找 [ ]
If [Get (查找个数) = 1]
转到布局 ["发货单详情"]
Else If [Get (查找个数) > 1]
转到布局 ["发货单"]
Else
显示自定义对话框 ["未找到记录。"]
End If
